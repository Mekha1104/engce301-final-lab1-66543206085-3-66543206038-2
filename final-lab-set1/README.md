## How to Run

1. Generate SSL certificates
./scripts/gen-certs.sh

2. Copy environment file
cp .env.example .env

3. Start services with Docker
docker compose up --build

Services will run on:

Auth Service → http://localhost:3001  
Task Service → http://localhost:3002  
User Service → http://localhost:3003

## How HTTPS Works in This System

HTTPS secures communication between the client and server using TLS encryption.

1. The client connects to the server using HTTPS.
2. The server sends its SSL certificate to the client.
3. The client verifies the certificate.
4. A secure encrypted connection is established.
5. All HTTP requests and responses are encrypted.

In this project, HTTPS is enabled using self-signed certificates generated by:
./scripts/gen-certs.sh

These certificates allow secure communication between services and clients during development.
